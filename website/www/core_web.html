CORE FUNCTIONS: WEB

This package exports several entry points into the JS environment, including:

  • web.run
  • web.stop
  • web.get
  • web.handler
  • web.url.parse

This API allows JS to fetch from the web and to create a web server.

WEB.RUN

web.run(addr);

Run a web server.

Example:


 var server = web.run(":http");

WEB.STOP

web.stop(server);

Stop a web server.

Example:


 var server = web.run(":http");
 web.stop(server);

WEB.GET

web.get(url);

Fetch an URL and return its contents.

Example:


 var html = web.get("http://www.cnn.com");

WEB.URL.PARSE

web.url.parse(url);

Parse and url and return its component parts.

Example:


 var url = web.url.parse("http://www.cnn.com");

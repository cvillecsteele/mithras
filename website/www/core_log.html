CORE FUNCTIONS: LOG

This package exports several entry points into the JS environment, including:

  • log
  • log0
  • log1
  • log2

This API allows resource handlers to log.

LOG

log(message);

Logs a message in the default format

Example:


  log("hello");

LOG

log(message);

Logs a message in the default format

Example:


  log("hello");

LOG0

log0(message);

Logs a more important message.

Example:


  log0("hello");

LOG1

log1(message);

Logs a less important message

Example:


  log1("hello");

LOG2

log1(message);

Logs an even less important message

Example:


  log2("hello");

 


 # CORE FUNCTIONS: USER


 

 This package exports several entry points into the JS environment,
 including:

 > * [user.lookup](#lookup)

 This API allows resource handlers to get system user ids.

 ## USER.LOOKUP
 <a name="lookup"></a>
 `user.lookup(user);`

 Get the user's UID.

 Example:

 ```

 user.lookup("root");

 ```

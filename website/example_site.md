  
 
 # Complete AWS Stack
 
 Usage:
 
     mithras -v run -f example/site.js
 
 This example builds a more-or-less complete and "typical" AWS
 stack, including:
 
 * VPC
 * Subnets
 * Security Group
 * IAM Instance Role
 * Elastic Load Balancer
 * DNS
 * RDS Database
 * Elasticache For Caching
 * S3 Bucket and Object
 * Keypair
 * Instances
 * Instance Configuration
   * Bootstrapping
   * Package Installation
   * A Git Repo
   * Nginx Setup
